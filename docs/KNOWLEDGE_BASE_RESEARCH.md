### 1.4 efaqa-corpus-zh (中文) ⚠️ 需授权

| 项 | 内容 |
|---|---|
| 出处 | 斯坦福 + UCLA + 台湾辅仁大学 + Chatopera |
| 规模 | **20,000 条** 中文心理咨询对话 (迄今最大公开中文) |
| 标签维度 | S1 烦恼类型 / S2 心理疾病 / S3 SOS (紧急度递进) |

| 评估 | **RAG 检索增强** 效果最好的中文心理数据集之一, 但商用需联系 Chatopera |

### 1.5 调研结论: 我们能怎么用

| 用途 | 推荐数据集 | 商业风险 |
|---|---|---|
| **AI prompt 优化 (英文参考)** | EmpatheticDialogues / MentalChat16K | 低 (内部训练参考) |
| **危机响应模式参考** | efaqa-corpus-zh (S3 SOS 标签) | 中 (商用需授权) |
| **RAG 检索知识库** | 暂用维基百科心理学条目 + 北京安定医院科普 | 低 |
| **危机词表基准** | PHQ-9 第 9 题 (有公开中文版) | 极低 (已发表学术) |

**v0.1 建议**: 先用 PHQ-9 / GAD-7 / 危机词表 + 6 疗法权威书摘要做知识库, 不直接用数据集 (避免授权风险)。

## 2. 心理评估量表 (公开发表 + 中文版)

### 2.1 PHQ-9 患者健康问卷 (抑郁) ✅ 公开

| 项 | 内容 |
|---|---|
| 出处 | Kroenke, Spitzer & Williams (2001), 已被翻译成 100+ 语言 |
| 中文版 | 多个机构发表过中文版, 信效度良好 |
| 评分 | 0-27 分, 0-4 无/最小, 5-9 轻度, 10-14 中度, 15-19 中重度, 20-27 重度 |
| **关键问题** | **第 9 题: "有不如死掉或用某种方式伤害自己的念头"** — 任何 > 0 分 → 危机响应 |
| 版权 | **公开免费使用, 非商业 / 商业均可** (Spitzer 公开授权) |
| 中文论文 | 司天梅 等 (北京大学第六医院), 2010, PSP-CH 中文版信效度 |
| 链接 | <https://www.phqscreeners.com/> (官方 PDF 下载) |

### 2.2 GAD-7 广泛性焦虑量表 ✅ 公开

| 项 | 内容 |
|---|---|
| 出处 | Spitzer, Kroenke, Williams & Löwe (2006) |
| 评分 | 0-21 分, 0-4 最小, 5-9 轻度, 10-14 中度, 15-21 重度 |
| 中文版 | 多个中文信效度论文, 标准版广泛使用 |
| 版权 | **公开免费使用** |
| 链接 | <http://www.phqscreeners.com/> |

### 2.3 中国心理危机评估 (本土化) ⚠️ 需整理

- **个体和社会功能量表 (PSP)** 中文版 (司天梅等, 2009/2010 发表) — 评估心理功能

### 2.4 MBTI ⚠️ 版权注意

| 项 | 内容 |
|---|---|
| **官方版权** | The Myers-Briggs Company (全球商业版权 + 商标) |
| **免费使用** | **不对个人开放, 仅授权 B 端** (Step I/II 单次测评数百上千元) |
| **可免费用的** | 16Personalities (www.16personalities.com/ch) - 融合大五人格, 国际通用 |

## 3. 危机响应 (12356 + 标准流程)

### 3.1 ✅ **12356 全国统一心理援助热线** (2025-05-01 启用)

| 项 | 内容 |
|---|---|
| **电话号码** | **12356** (短号码, 易记) |
| **设立方** | 国家卫生健康委员会 + 工业和信息化部 |
| **启用日期** | 2025-05-01 (31 个省全部开通) |
| **服务时间** | **24 小时** |
| **覆盖** | 心理健康教育 / 心理咨询 / 心理疏导 / **心理危机干预** |
| **接听方** | 各设区市卫生健康行政部门, 精神科医师 + 心理治疗师 |
| **典型案例** | 北京安定医院: 救下跳河自杀未遂青年 |
| **接听规模** | 北京 5 天接听 500 通, 全国 50 万通 (截至 2025-07) |
| **链接** | <https://www.nhc.gov.cn/> |
| **建议** | **小程序所有危机弹窗 + 24h 卡片, 必须显示 "12356", 不要再列地区号码** |

### 3.2 危机响应标准流程 (基于国际 + 国内实践)

```
[用户输入危机关键词]
    ↓
[AI 检测层] (1 秒, 关键词触发)
    ↓
[分流决策]
    ├─ S3 紧急 (自杀/自伤正在发生) → 立即弹 12356 + 110 联动
    ├─ S2 中度 (PHQ-9 ≥ 15 / 持续表达"想消失") → 弹 12356 + 引导预约
    └─ S1 轻度 (焦虑/失眠/孤独) → AI 共情 + 主动引导预约
    ↓
[同步记录] (不阻塞用户)
    ├─ crisis_logs 云数据库 (脱敏)
    └─ 危机预警广播 → 区域经理 (B 端) / 运营 (C 端)
    ↓
[后续回访] (24h / 72h)
    ├─ AI 主动跟进: "昨天聊到的 XX, 现在怎么样了?"
    └─ 高危: 真人咨询师电话回访
```

### 3.3 危机关键词参考清单 (基于 PHQ-9 / C-SSRS 翻译)

```
S3 紧急 (立即响应):
  - 自杀 / 自残 / 跳楼 / 跳河 / 上吊 / 割腕
  - "我想死" / "不想活了" / "活着没意思" / "一了百了"
  - "有具体计划" / "准备好工具"

S2 中度 (主动引导预约):
  - "很痛苦" / "受不了" / "崩溃" / "想消失"
  - 持续 2 周以上的: 失眠 / 暴食 / 嗜睡 / 注意力涣散
  - PHQ-9 第 9 题任何 > 0

S1 轻度 (AI 共情):
  - "焦虑" / "压力" / "孤独" / "没人理解"
  - 情绪波动 / 工作压力 / 人际关系 / 失恋
```

**调研空白**: 国内**没有权威统一的危机词表**, 我们需要自己整理 (基于 C-SSRS 中文版 + PHQ-9 + 临床经验), 整理后让合作咨询师 review。
