## 一、文献/市场调研

### 1.1 MBTI (国外成熟, 16personalities/赛可心理)
- **2025 调研**: MBTI 200 题完整版占主流 (替代 12/28 题速测), 用户追求**准确度**, 不再满足"娱乐化"
- **应用场景**: 职业规划 / 团队沟通 / 自我认知 / 婚恋匹配 / **个性化推荐**
- **国内合规**: MBTI 不是临床诊断工具, 必须标"参考"
- **4 维度** × 16 类型: I/E (能量) / S/N (信息) / T/F (决策) / J/P (生活)

### 1.2 星盘 (西方, Co-Star/Astrology.com)
- **2024 调研**: 西方年轻女性 (18-35) **Z 世代**主要使用
- **3 大产品**: Co-Star ($7/月订阅), Sanctuary (免费+内购), CHANI (BIPOC 友好)
- **数据基础**: 占星术 + 行星位置 (NASA 开源数据免费)
- **关键设计**: **每日推送** + **社交** (朋友对比) + **仪式感** (UI 设计)
- **中国限制**: 部分占星内容 (如"行星逆行"对中国用户文化认同低), 需本土化

### 1.3 五行八字 (中国本土, 命理/中医)
- **2026 调研**: 5.6 亿中国人接触过命理/八字, 90 后/00 后"赛博算命"流行
- **算法基础**: 天干地支 (10 天干 × 12 地支 = 60 甲子) + 五行 (金木水火土)
- **关键 4 步**:
  1. 把生日 (年月日时) 转八字 (4 柱 × 2 字 = 8 字)
  2. 五行属性分类
  3. 五行生克 (金生水/水生木/木生火/火生土/土生金; 金克木/木克土/土克水/水克火/火克金)
  4. 推导性格 + 推荐
- **文化心理**: "五行缺金" → 用户会**主动找补** (穿金饰/金器) — 这就是**消费驱动力**
- **国内合规**: 命理 App 必须标"文化娱乐", 不能标"预测/算命"

### 1.4 推荐系统 (学术 + 工程)
- **协同过滤 (CF)**: "和你类似的人喜欢这个" — Netflix 经典
- **内容过滤 (CB)**: "基于你的属性, 推荐匹配" — Pandora 音乐
- **混合 (Hybrid)**: CF + CB + 知识图谱 — Spotify, Amazon
- **冷启动问题**: 新用户没数据 → **必须用属性** (MBTI/星盘/五行) 兜底
- **2024 新趋势**: LLM 增强推荐 (用 GPT 生成解释/总结)

## 四、产品设计 (我建议)

### 4.1 名称 (我建议几个)
- **「心选」(XinXuan)** — 心智选择, 简练
- **「知荐」** — 知道 + 推荐
- **「象限」** — MBTI 是 4 象限, 有点玄
- **「五行星」** — 直接点题
- **「拾光」** — 时间/光, 文艺

**MVP 建议**: 「心选」, 后可改名

### 4.2 MVP 流程 (先做书单)
```
1. 用户注册: 生日 + 出生时间 + 性别 (3 个必填)
2. 测 MBTI: 28 题简版 (2 分钟, 可跳过)
3. 选星盘: 自动算 (基于生日, 不需要输入)
4. 算五行: 自动算 (基于八字, 不需要输入)
5. 出"三合一画像":
   - MBTI: ISFP / 五行: 木旺 / 星盘: 金星双鱼
   - "你天生对美敏感, 喜欢独处, 但需要补充金元素"
6. 推荐 5 本书:
   - 每本 1 句话解释为什么
   - "《小王子》适合你, 因为你像小王子一样..."
7. 用户标记喜欢/不喜欢
8. 学习 (reading-fl Emotion Head 升级) → 个性化更好
```

### 4.3 后端架构 (基于你现有)
```
cloudfunctions/
  - user/         # 存用户画像 (MBTI/星盘/五行 + 历史)
  - book/         # 书库 (豆瓣 API 或自爬)
  - recommend/    # 推荐核心 (混合 CF + CB)
  - explain/      # 用 LLM 生成"为什么推荐这本书" (P1 RAG)
```

### 4.4 前端 (微信小程序, 复用 qi_wechat)
- 5 个 page: 测一测 / 我的画像 / 推荐 / 书单详情 / 反馈
- 风格: 简洁, 大色块, 中国风 (五行配色: 金白/木青/水黑/火红/土黄)

### 4.5 数据来源 (国内合规)
- **MBTI 题库**: 16Personalities 中文版 (公开) 或 28 题简版
- **星盘算法**: 瑞士星历表 (NASA JPL 公开数据, 自己算)
- **八字算法**: 农历转换 + 五行生克 (开源 Python 库, 如 `lunardate` + `bazi`)
- **书库**: 豆瓣读书 API (免费但有限流) 或自爬 (1 万本够 MVP)

## 六、合规 + 风险 (重要)

### 6.1 命理合规
- 国内命理/算命 App 必须标"**文化娱乐**" 不是"预测/算命"
- 不能用"算命/预测/财运" 关键词
- 不能收钱承诺"命运/改运"
- **建议**: 标"基于心理学 + 中国传统文化的性格画像"

### 6.2 MBTI 合规
- 标"心理学参考" 不是"诊断"
- 16Personalities 商用前要看 license
- 建议自建题库 (28 题中国修行版, 公开有)

### 6.3 隐私
- 生日是个人敏感信息, **必须脱敏存储**
- 命理/星盘**有宗教/文化敏感性**, 部分用户不想被记录
- 建议: 用户**可删除自己的所有画像数据**
