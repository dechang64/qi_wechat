## 什么是焦虑?

焦虑是一种正常的情绪反应 — 面对不确定 / 危险时的"警觉信号".

焦虑本身不是病, **过度焦虑才是**. 当焦虑:
- 持续 6 个月以上
- 与实际威胁不成比例
- 影响工作 / 学习 / 关系
- 身体症状明显 (心跳快 / 出汗 / 失眠 / 肌肉紧绷)

可能就构成**焦虑障碍**.

## 焦虑的种类

- **广泛性焦虑障碍 (GAD)**: 长期担心各种事
- **惊恐障碍 (Panic Disorder)**: 突然强烈的恐惧感, 类似心脏病发作
- **社交焦虑障碍 (Social Anxiety)**: 害怕被评判 / 公开讲话
- **特定恐惧症 (Phobia)**: 对某具体事物 (飞行 / 高度 / 注射)
- **强迫症 (OCD)**: 反复的想法 + 行为 (检查 / 洗手)

## 7 个身体症状 (焦虑的"伪装")

1. 心跳快 / 心悸
2. 出汗 / 手脚冰凉
3. 胸口闷 / 呼吸短促
4. 头晕 / 颤抖
5. 胃部不适 / 恶心
6. 肌肉紧绷 / 肩膀酸痛
7. 失眠 / 难以集中

很多焦虑患者先去看心脏科 / 消化科, 检查没问题后才意识到是焦虑.

## GAD-7 自评量表 (7 题)

过去 2 周, 你被以下问题困扰的频率?

| 题 | 内容 |
|---|---|
| 1 | 感觉紧张, 焦虑或急切 |
| 2 | 不能够停止或无法控制担忧 |
| 3 | 对各种各样的事情担忧过多 |
| 4 | 很难放松下来 |
| 5 | 由于不安而无法静坐 |
| 6 | 变得容易烦恼或急躁 |
| 7 | 感到似乎将有可怕的事情发生而害怕 |

每题 0-3 分 (完全不会=0, 几天=1, 超过一周=2, 几乎每天=3), 总分 0-21.

**评分标准**:
- 0-4: 最小焦虑
- 5-9: 轻度
- 10-14: 中度
- 15-21: 重度

**中度及以上建议看医生**, 重度需要专业治疗 (CBT + 必要时药物).

## 5 个自助方法

### 1. 腹式呼吸 (1 分钟)
吸气 4 秒 → 屏气 4 秒 → 呼气 6 秒, 重复 5-10 次.

### 2. 5-4-3-2-1 grounding
说出 5 个看到的, 4 个听到的, 3 个摸到的, 2 个闻到的, 1 个尝到的.

### 3. 渐进式肌肉放松
从脚趾到头顶, 依次绷紧 5 秒再放松.

### 4. 写下焦虑
把担心的事写下来, 然后问"最坏会怎样? 发生了能应对吗?"

### 5. 限制咖啡因 / 酒精
咖啡因会让焦虑加剧, 酒精会让焦虑反弹.

## 何时一定要看医生

- GAD-7 ≥ 10 (中度及以上)
- 焦虑已严重影响工作 / 学习 / 关系
- 有自伤 / 自杀念头 — **立即拨打 12356**
